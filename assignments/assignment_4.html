Assignment 4: Wi-FI Programming Lab - MQTT LAB pt1

This assignment focuses on programming Wi-Fi capabilities using ESP-IDF and using MQTT protocol.

1. Activity Goals

The main goals of this activity are:

  • Understand the basics of MQTT protocol
  • Implement MQTT client functionality in ESP-IDF
  • Connect to an MQTT broker to read and publish messages
  • Control a DC motor using MQTT
